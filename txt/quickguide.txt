El símbolo al principio de una línea de comandos indica:
  $ = hacer la sentencia como usuario
  # = hacer la sentencia como administrador

El símbolo ~ representa el directorio del usuario, es decir, /home/usuario, si se muestra en la
línea de comandos, significa que las operaciones se están realizando en el home del usuario
  ~$

// ------------------------------------- Guía rápida [BEGIN] ------------------------------------ //

1. Crear el proyecto
  $ django-admin startproject mysite



2. Crear la aplicación
  $ cd mysite
  $ python manage.py startapp polls

3. Escribir la primera vista: mysite/polls/views.py

from django.http import HttpResponse


def index(request):
    return HttpResponse("Hello, world. You're at the polls index.")

4. Señalar la url raíz de polls editando mysite/urls.py

from django.conf.urls import include, url
from django.contrib import admin

urlpatterns = [
    url(r'^polls/', include('polls.urls')),
    url(r'^admin/', admin.site.urls),
]

5. Correr el servidor
  $ python manage.py runserver

6. Crear una base de datos en PostgreSQL y conectarla con Django
editando mysite/settings.py

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'nombrebasededatos',
        'USER': 'user',
        'PASSWORD': 'password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}

7. Primera migración
$ python manage.py migrate

8. Primer modelo: mysite/polls/models.py

from django.db import models


class Question(models.Model):
    question_text = models.CharField(max_length=200)
    pub_date = models.DateTimeField('date published')


class Choice(models.Model):
    question = models.ForeignKey(Question, on_delete=models.CASCADE)
    choice_text = models.CharField(max_length=200)
    votes = models.IntegerField(default=0)

9. Activar el modelo: mysite/settings.py

INSTALLED_APPS = [
    'polls',
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
]

10. Migrar polls
  $ python manage.py makemigrations polls
  $ python manage.py migrate
